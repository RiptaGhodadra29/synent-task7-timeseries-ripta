# 📈 Stock Price Time Series Analysis

## 🚀 Internship Task
This project is part of the Synent Technologies Data Science Internship Program.

---

## 📌 Problem Statement
The objective of this project is to perform time series analysis on stock price data to identify trends, detect seasonal patterns, and derive meaningful insights.

---

## 📊 Dataset

The dataset used in this project is a Stock Price dataset.

🔗 Dataset Link: https://www.kaggle.com/datasets/prince7489/stock-market-dataset

Note: Dataset is not uploaded due to size limitations.

## ⚙️ Approach

### 1. Data Preprocessing
- Converted `Date` column to datetime format
- Removed invalid/missing values
- Set `Date` as index for time series analysis
- Filtered a single stock for accurate analysis

---

### 2. Trend Analysis
- Visualized stock price over time using line plot
- Identified overall upward/downward trends

---

### 3. Moving Average
- Applied 30-day moving average
- Smoothed short-term fluctuations
- Highlighted long-term trend

---

### 4. Seasonality Detection
- Extracted month from date
- Calculated average price per month
- Visualized monthly variation

---

### 5. Volume Analysis
- Analyzed trading volume over time
- Identified spikes indicating high activity

---

## 📈 Insights
- Stock prices show clear trends over time
- Moving average reveals long-term direction
- Certain months exhibit higher average prices (seasonality)
- Volume spikes indicate increased market activity

---

## 🧠 Conclusion
Time series analysis helps understand stock behavior over time and supports better decision-making in financial analysis.
